Usando Inspetor de Elementos4

(voltar)

O Inspetor de Elementos é uma ferramenta poderosa disponível na maioria dos navegadores modernos, como Google Chrome, Firefox, Microsoft Edge e Safari.
Uma forma de abrir o inspetor de elementos é clicando com o botão direito do mouse em qualquer parte da página web e selecionando "Inspecionar" ou "Inspecionar Elemento" no menu de contexto.
Outra forma é pressionando a tecla F12 ou Ctrl+Shift+I (Cmd+Option+I no macOS) no teclado.
O Inspetor de Elementos permite que você visualize e edite o código HTML e CSS de uma página web em tempo real.
Você pode clicar em qualquer elemento da página para ver seu código correspondente, incluindo tags HTML, atributos e estilos CSS aplicados.
Além disso, o inspetor de elementos oferece várias funcionalidades úteis. Nesta ferramenta você tem vários recursos para estudar sobre como aquela página foi criada/como foi feita.

Estrutura básica do HTML5

(voltar)

Primeiramente, você cria uma pasta na área de trabalho para onde vão seus projetos e depois. No VsCode você crie uma pasta
a onde vai ficar seus projetos, fazendo o seguinte:

  • Arquivo;
  • Abrir pasta;
  • Seleciona a pasta que você quer armazenar seu projeto;
  • Ele vai aparecer pra você toda a estrutura de arquivo, pra você do lado esquerdo;
  • Depois você clica em novo arquivo;
  • Coloca o nome do arquivo com a extensão .html;
  • Clica em enter;
  • Pronto você já tem seu arquivo HTML criado.

A estrutura básica do HTML é composta pela declaração:! DOCTYPE html, que informa ao navegador que o documento é um arquivo HTML5. O HTML consiste em básicamente, três partes principais:
1. Tudo que estiver entre (html) a abertura e o (/html) fechatemen o Sistema vai entender que é um elemento HTML.
2. Cabeçalho (Head):Contém informações sobre o documento, como o título da página, metadados e links para arquivos externos (como folhas de estilo CSS ou scripts JavaScript).
No (head)temos uma tag chamada (title) que é onde colocamos o título da nossa página, que aparece na aba do navegador quando a página é aberta.
3. Corpo (Body):Contém o conteúdo visível da página, como texto, imagens, links, listas e outros elementos.
Observações: Toda vez, que aparecer na parte superior onde esta escrito o nome do arquivo index.html, uma bolinha branca/preta significa
que você precisa salvar o documento clicando em:CTRL + S. Toda vez que você queser ver o que você esta criando click em:
Explorador, depois clica com o botão direito do mouse em cima do arquivo que você criou e clica em: Abrir com Open with Live Server.
Pronto, o seu navegador vai abrir e você vai ver o que você esta criando, o Live Server vai sempre atualizar pra você não havendo a necessidade de ficar atualizando a página.
Se você não tiver o Live Server instalado, você pode clicar em: Extensões, depois digitar na barra de pesquisa: Live Server, clicar em instalar e pronto.
Depois de instalado você pode clicar com o botão direito do mouse em cima do arquivo que você criou e clicar em: Abrir com Open with Live Server.
Pronto, o seu navegador vai abrir e você vai ver o que você criou.

  • A Tag (img/) também não tem fechamento, ela já faz isso, observe que diferente das outras tags ela tem uma / no final. Ela tem um atributo que é o

    Vamos falar das principais tags que usamos em textos,sendo a primeira dela 'h1'.Começando pelo que significa do h: head ou seja é cabeçalho ou título da sua página.
    No 'h1'você tem o título principal, no 'h2' o sub-título e as demais são sub, sub-titulo vai de acordo com a posição de cada um,
    como do maior que é 'h1' para o menor de todos que é o 'h6', cada titulo ou sub-titulo o sistema vai diminuindo a letra de acordo com a importancia de cada 'h'.
    Você já pode imaginar como vai ficar cada titulo ou sub-título de pendendo de qual 'h' você esta usando.O navegador entende que o h1 é o titulo
    e o h2 é o sub-titulo dessa forma ele coloca um espassamento entre eles, você pode alterar tudo isso depois via 'CSS' O 'h' é voltado para
    título enquanto que o corpo do texto é definido pelo (p) de paragrif ou parágrafo, sendo assim a cada parágrafo,ou ponto final você coloca um (p) novo.
    Dentro do seu texto você quer fazer uma citação, frase de efeito ou destacar algo mais dentro do texto, você tem uma tag chamada (blockquote) com ela você acrescenta uma frase, um exemplo de um
    outro autor, uma poesia algo que você ache interesante dentro da quele assunto, nessa hora o navegador da uma margem para mostrar que aquele é uma citação.
    (Blockquote) é a tag que em HTML serve para indicar uma seção de texto que é uma citação de outra fonte, geralmente recuando o texto para diferenciá-lo do conteúdo principal da página.
    No corpo do texto você pode destacar uma palavra ou até mesmo uma frase usando tag strong para deixar em italico 'u' para deixar sublinhado, você também pode usar um marcador de texto usando para isso a tag (mark), na tag (sup) por exemplo você pode elevar uma letra ou número acima de qualquer palavra é como se fosse elevar um simbolo para destacar um título por exemplo:
    #DIO.me#

    Listas ordenadas e não ordenadas9

    (voltar)

    Vamos ver agora sobre as listas ordenadas e não ordenadas, começando pela ordenada assim temos a tag (ol) que seria lista ordenada em inglês, dentro dessa lista
    ordenada não podemos colocar um texto diretamente, para isso devemos colocar uma outra tag que é chamada de (li) [list item: item da lista] que
    essa tag na realidade é chamada de tag filha por esta dentro de outra tag, você vai colocar os itens um a baixo do outro. Como é uma lista ordenada ele
    vai entender que você vai colocar na ordem,ou seja 1, 2, 3,... e assim por diante sendo também possível colocar dentro dessa lista uma outra lista, por exemplo você pode colocar primeiro uma lista ordenada e dentro da mesma você abrir uma outra lista usando a tag (ul) que se refere a lista
    não ordenada:{ol, li,li,li,ul,li,li,ul,li,ol}. A diferencia entre uma lista ordenada e não ordenada é que: a ordenada ela
    vem com números ordenados como 1,2,3,... e a não ordenada ela vem com marcadores, o navegador vai entender que é não ordenada
    ou seja não tem uma ordem. Essa lista pode ser udasa dentro do texto para um sumário, menus entre outras listas.

    Links10

    (voltar)

    Os links são criados usando a tag (a) que significa ancorar, ou seja, você esta ancorando em algum lugar, essa tag é acompanhada do atributo (href) que é o endereço
    para onde você quer ser direcionado, esse endereço pode ser uma página web, um arquivo, um endereço de e-mail, uma ligação na mesma página ou um endereço na URL.
    O href é um atributo exclusivo da tag "a" é a onde você vai clicar e ser direcionado para uma outra página, dentro desse atributo tem a (Target) que te da algumas opções: freme, self, blank.
    O target serve para definir onde o link será aberto.A palavra link se refere a ligação entre duas páginas ou recursos na web.
    Existem diferentes tipos de links, como links internos (dentro do mesmo site), links externos (para outros sites), links de ancoragem (para seções específicas dentro de uma página) e links de e-mail (para abrir o cliente de e-mail com um endereço pré-preenchido).
    Além do href você pode usar um outro atributo, que é o title: escreve o que você quer que apareça ao passar o mause na palavra em destaque, você adciona ela antes da tag (target).
